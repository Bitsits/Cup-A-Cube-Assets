  • What is this program about?
    • This program is about unlocking the potential of developers in India, like you. Register, build and submit one or more Windows Phone Apps on App Hub. If they get certified and published at the Windows Phone Marketplace by 31st March, 2012, you are entitled to receive a Windows Phone device to test your future apps.
  • Can I participate in this program?
    • If you are a technology professional – Software Developer, Project Manager or Software Architect – or a technology enthusiast working and residing in India, you are eligible to participate in this program. The full eligibility criteria are available here.
  • How will I benefit from participating in this program?
    • There are multiple benefits on offer for you:

      • If you submit three (3) Windows Phone Apps that get listed at the Marketplace within the program period, then you qualify to receive a
        Windows Phone test device (model and manufacturer to be decided by Microsoft).

      • If you submit two (2) new Windows Phone Apps – that you have ported from existing Android and/or iPhone apps – which get listed at the
        Marketplace within the program period, then you qualify to receive a Windows Phone test device (model and manufacturer to be decided by
        Microsoft).

      • If you are a woman developer, and submit one (1) new Windows Phone app – based on entertainment, fashion, leisure, gaming, recreation or
        travel themes – which gets listed at the Marketplace within the program period, then you qualify to receive a Windows Phone test device (model and
        manufacturer to be decided by Microsoft).

  • What do I need to do to submit my app at the Windows Phone Marketplace?
    • You must be registered as an App Hub member in order to submit an app for Windows Phone. App Hub membership fee is USD 99 per year. Paid membership enables you to publish and manage applications at Windows Phone Marketplace, and to post to the App Hub forums to get support from the developer community.

      You must have a Windows Live ID to register. If you don’t have one, go to Windows Live and create one.

      More questions about Windows Phone app and game submissions
  • What is App Hub?
    • App Hub is a vibrant community of apps and game developers for Windows Phone and Xbox LIVE Indie Games on Xbox 360. As part of App Hub membership, you get a developer dashboard so you can manage all aspects of how your app or game appears in Marketplace, monitor downloads, and track how much money you’ve earned if you charge for your work.
  • What is the Windows Phone Marketplace?
    • Once you’ve developed your app or game, Windows Phone Marketplace is where you can sell or give away your creation for free to the global audience of Windows Phone owners.
  • What happens after I submit my app on App Hub? How long does the certification process take?
    • After an app is submitted on App Hub, it will be tested to verify whether it adheres to the certification guidelines. If it passes certification testing, then it will be published on the Windows Phone Marketplace. The duration of the certification process is variable, but it usually takes around one to four weeks.
  • Do I need to get my app certified before it’s published at the Windows Phone Marketplace?
    • All applications must pass certification testing before being published to Windows Phone Marketplace. You can find the application policies and technical certification requirements in Windows Phone 7 application certification requirements. There’s no cost for app/game certification or application updates.

      More questions about Windows Phone app and game certification
  • Will I get multiple test devices for multiple apps published on Marketplace during the program period?
    • A participant will be eligible to get only one (1) test device as part of this program, regardless of whether he/she qualifies in more than one category.
      Also note that these Windows Phone applications must be distinct and should have non-trivial utility value for the application user. In case the submitted applications are found to be “Spam” or close replicas of existing applications, these will not qualify for the program.
  • How will I know if my app qualifies?
    • Results will be announced in this website on a monthly basis.
